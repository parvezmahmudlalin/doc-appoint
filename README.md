# DocAppoint

DocAppoint is a doctor appointment booking platform that helps users find doctors and schedule appointments easily and efficiently.

Live Link:

https://doc-appoint-orcin.vercel.app

Features:

* Search doctors by specialty
* Book appointments quickly and easily
* Secure user authentication system
* View doctor profiles and availability
* Responsive design for all devices
* Simple and user-friendly interface

Technologies:

* React.js
* HeroUI
* BetterAuth
* React-Icons
* GravityUI-Icons
* Tailwind CSS
* Firebase
* Node.js
* Express.js
* MongoDB
